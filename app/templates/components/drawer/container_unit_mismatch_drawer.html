Let’s Get This Fixed
  1. Confirm the recipe yield unit. Make sure the predicted yield is expressed in the unit you actually bottle or package (e.g., fl oz for liquids, g for lipstick fill weights).
  2. Align container capacity units. Containers must have a capacity unit that can convert to the recipe yield unit. Update the inventory item’s capacity unit (e.g., use fl oz instead of count) or add containers that already match.
    Example: Lipstick tubes measure grams, so the recipe should yield a weight. Cosmetics jars measure fluid ounces, so the recipe should yield a volume.
  3. If you need to move forward now, disable containers for this batch. Uncheck “Requires Containers” in Plan Production to proceed without auto-fill.
